🚀 ComunicadorEterna
El ComunicadorEterna es una solución tecnológica diseñada con el corazón para facilitar la comunicación a personas con limitaciones de movilidad.

✨ ¿Por qué este proyecto?
Este proyecto nace de la necesidad de crear un puente accesible y rápido. Hemos implementado un teclado web optimizado con control de velocidad de texto (text speed), permitiendo que la comunicación sea fluida, personalizable y, sobre todo, humana.

🛠️ Tecnologías y Características
Web Development: Estructura limpia, ligera y accesible.

Optimización de velocidad: Ajustes en tiempo real para adaptarse al ritmo del usuario.

Diseño centrado en el usuario: Interfaz intuitiva y fácil de gestionar.

Versatilidad: Incorpora distintos teclados con diversas funcionalidades para adaptarse a diferentes capacidades o limitaciones.

Ligereza técnica: Se puede descargar y ejecutar como aplicación local en cualquier ordenador. Al ser un sistema muy liviano, puede operar incluso en equipos antiguos sin sufrir lag ni bloqueos.

Requisito: Requiere conexión a internet para su funcionamiento.
